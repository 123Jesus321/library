Как добавить свои PDF-книги:
1) Скопируйте PDF-файлы в эту папку (knigi), например: my-book.pdf
2) Откройте файл books.js
3) Добавьте новую запись в массив books:
   {
       id: "my-book",
       title: "Моя книга",
       author: "Автор",
       pdf: "my-book.pdf"
   }
4) Сохраните файл и обновите страницу index.html
